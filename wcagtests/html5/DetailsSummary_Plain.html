Examples

Example without the open attribute

The example below should render with a disclosure widget next to "Knot", and the widget should be collapsed.

Knot

A knot is a unit of speed equaling 1 nautical mile per hour (1.15 miles per hour or 1.852 kilometers per hour).

The above example

  • Works in Google Chrome 19.0.1084.52 m on Windows XP: both with the mouse and the keyboard; it is possible to tab to "Knot" and open and close the widget with Enter.
  • Works in Comodo Dragon 19.0.1084.56 on Windows 7: both with the mouse and the keyboard; it is possible to tab to "Knot" and open and close the widget with Enter.
  • Does not do anything in Internet Explorer 8.0.6001.18702 on Windows XP or Internet Explorer 9.0.8112.16421 in Windows 7; no keyboard access; in addition, neither the details nor the summary cause a new line.
  • Does not do anything in Firefox 12.0 on Windows XP and Windows 7; no keyboard access; in addition, neither the details nor the summary cause a new line.
  • Does not do anything in SeaMonkey 2.9.1 on Windows XP or SeaMonkey 2.10 on Windows 7; no keyboard access; in addition, neither the details nor the summary cause a new line.
  • Does not do anything in Opera 11.64 (build 1403) on Windows XP and Windows 7; no keyboard access (the shortcuts Tab, Ctrl + ↓, Ctrl + ↑, Shift + ↓, Shift + ↑ cannot be used to reach the summary); in addition, neither the details nor the summary cause a new line.
  • Does not do anything in Safari 4.0.4 on Windows XP or Safari 5.1.7 on Windows 7; no keyboard access. (What keys should be used in Safari on Windows for keyboard navigation? The tab key does not cycle through links.)

Note: Google Chrome, Comodo Dragon (which is based on Chromium) and Safari render details as a paragraph (or block-level element in HTML 4.01 parlance).

Example with the open attribute

The example below should render with a disclosure widget next to "Knot", and the widget should be expanded.

Fork

(Chess): A knot is a move that uses one piece to attack two or more of the opponent's pieces simultaneously, hoping to achieve material advantage, since the opponent can only counter one of the two threats. (Source: Wikipedia: Chess tactic.)

The above example leads to the same results as the example without the open attribute.

Note: the second example is not keyboard accessible in Google Chrome and Comodo Dragon (on Windows 7) unless the first example is commented out or removed!?
