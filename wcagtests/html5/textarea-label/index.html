Test Pages for textarea and label

This page contains examples of textarea elements with label elements, sometimes explicitly connected with each other, sometimes not. Question: If the label element is wrapped around the textarea element, does the textarea element's content become part of the accessible name? This is not entirely obvious when reading the accessible name and description computation algorithm. In addition, implementation in screen readers may vary.
