This page uses the following a elements:


    <a href="index.html" rel="top">Home page</a>
    <a href="link-rel-4.html" rel="prev">Page Four</a>
    <a href="link-rel-6.html" rel="next">Page Six</a>
    <a href="link-rel-8.html" rel="last"></a>

For the first link in the above list, SeaMoneky displays the following link properties (choose Properties in the link's context menu):

The dialog displays the following link properties:

  • Address: (…)/wcagtests/html5/head-link-rel/index.html: the link's target.
  • Will open in: Same window (i.e. not in a different window, tab or frame).
  • Relation: top: this matches the value in the rel attribute.
