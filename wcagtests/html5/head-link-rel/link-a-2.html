Page Six — Test Site for the rel Attribute

Test for the rel attribute on the a element (inside body).

  • Home page
  • Christophe Strobbe (author of this site)
  • Last page

This page uses the following a elements:

    <a href="index.html" rel="top">Home page</a>
    <a href="https://github.com/cstrobbe" rel="author">Christophe Strobbe (author of this site)</a>
    <a href="link-a-4.html" rel="last">Last page</a>

Note that last is not a linktype acknowledged by the HTML 5.2 specification.
